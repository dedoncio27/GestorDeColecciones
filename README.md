# Gestor de Colecciones

Aplicación fullstack para la gestión de colecciones personalizadas.

## Enlaces del Proyecto

- **Tablero de Trabajo (Trello/GitHub Projects):** [Enlace al tablero]

## Estructura del Proyecto

- `src/`: Frontend en React + TypeScript
- `server/`: Backend en Node.js + Express
- `docs/`: Documentación del proyecto

## Instalación

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm start
```
